# 补充
使用  12MHz、定时器T1

## 方式0（定时5ms）
```
TMOD=0x00;//定时器T1工作在方式0
/*********TMOD(Timer Mode)**********/

TR1=1;//启动定时器T1
/*********TCON(Timer Control)即可以位操作，也可以字操作**********/

TH1=(8192-5000)/100;//高位取前两位，所以是除    或者  TH1 = (8912 - 5000) / 32
TL1=(8192-5000)%100;//低位取后两位，所以是取模  或者  TL1 = (8912 - 5000) % 32
/*********对定时器高八位和低八位赋初值**********/

```
---
## 方式1（定时50ms）
```
TMOD=0x10;//定时器T1工作在方式1
/*********TMOD(Timer Mode)**********/
/*********0001 0000**********/

TR1=1;//启动定时器T1
/*********TCON(Timer Control)即可以位操作，也可以字操作**********/

TH1=0x3c;//高八位  或者  TH1 = (65536 - 50000) / 256
TL1=0xb0;//低八位  或者  TL1 = (65536 - 50000) % 256
/*********对定时器高八位和低八位赋初值**********/
/*********50ms=50000us，所以(65536-50000)就是要给寄存器的初值**********/

```
## 方式2（定时100us）
```
TMOD=0x20;//定时器T1工作在方式2
/*********TMOD(Timer Mode)**********/
/*********0010 0000**********/

TR1=1;//启动定时器T1
/*********TCON(Timer Control)即可以位操作，也可以字操作**********/

TH1=0x9c/;//高八位
TL1=0x9c;//低八位
/*********对定时器高八位和低八位赋初值**********/
/*********100us就是100个机器周期，所以(256-100)就是要给寄存器的初值**********/

```
