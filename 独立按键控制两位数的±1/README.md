# ✏️独立按键控制加减✏️

**代码预览：**
```
unsigned int num = 0;
while(1)
{
	if(key_add == 0)                        //判断按键是否按下
	{
		delay(1250);                    //消抖
		if(key_add == 0)                //再次判断按键是否按下
		{
			while(key_add == 0)     //等待按键释放
			{
				display(num);
			}
			delay(1250);
			num++;
			if(num==100)            //我们只控制两位数的显示，当到达99时，下一位归零
			{
				num = 0;
			}
		}
	}
	display(num);                           //避免只有按下时才显示的情况
 }
```

---

## 分段解释：
**按键部分：**
```
if（key_add == 0)         //判断按键是否按下了
{                         //
  delay(1250);            //进行延时，消除按下抖动
    
  if(key_add == 0)        //确认按键按下
  {                       //已经按下了，执行以下 
    while(key_add == 0)   //等待按键释放
    {                     //在等待工程中的行动
      display(num);       //显示数字
    }      
   delay(1250);           //进行释放消抖
   num++;                 //!!!进行加法运算!!!在这时候才是真正完成一次"按下"的动作
   }//确认按下的if
}//初始判断的if
```
